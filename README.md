# AI Drug Discovery Pipeline (Multi-Biomarker)

## 🚀 Project Overview
This project implements a machine learning pipeline for biomarker-based drug discovery using the ChEMBL database. It extracts bioactivity data, processes it, and prepares it for predictive modeling.

## ⚙️ Features
- Data extraction from ChEMBL
- Target-based biomarker search
- Bioactivity analysis
- Data preprocessing for ML

## 🧠 Technologies Used
- Python
- Pandas
- ChEMBL Web Resource Client

## ▶️ How to Run
```bash
pip install pandas chembl_webresource_client
python drug_discovery_pipeline.py
